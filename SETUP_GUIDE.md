## 📋 ЧТО НУЖНО СДЕЛАТЬ:

### 1. Скачай файлы (если ещё не скачал):
- ✅ `setup_samurai.ps1` (скрипт автоустановки)
- ✅ `uzbek_strings.ftl` (узбекский язык)
- ✅ `uzbek_announcements.ftl` (узбекские объявления)

### 2. Распакуй samurai-main.zip на Desktop

### 3. Скопируй скачанные файлы:
```
Скопируй все 3 файла в папку:
C:\Users\user\Desktop\samurai-main\
```

Должно получиться:
```
samurai-main/
├── setup_samurai.ps1          ← НОВЫЙ
├── uzbek_strings.ftl          ← НОВЫЙ
├── uzbek_announcements.ftl    ← НОВЫЙ
├── bot.py                     ← УЖЕ ЕСТЬ
├── config.toml                ← УЖЕ ЕСТЬ
└── ... (остальные файлы)
```

## 🚀 ЗАПУСК (1 команда!):

### Открой PowerShell:

1. Нажми `Win + X`
2. Выбери "Windows PowerShell (Admin)" или "Terminal (Admin)"
3. Перейди в папку:
   ```powershell
   cd C:\Users\user\Desktop\samurai-main
   ```

4. Разреши выполнение скриптов:
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
   Нажми `Y` и Enter

5. Запусти скрипт:
   ```powershell
   .\setup_samurai.ps1
   ```

**ВСЁ! Скрипт сделает всё сам!** ☕
