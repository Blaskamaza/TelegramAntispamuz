# InglizchaAI - AI English Tutor for Uzbekistan

## Overview
A Telegram-based AI English tutor designed for the Uzbek market. It provides voice-to-voice conversation practice, grammar correction, and gamification.

## Tech Stack
- **Python 3.11+**
- **Aiogram 3.x:** Asynchronous Telegram Bot framework.
- **FastAPI:** Handles Webhooks and Payment callbacks.
- **Supabase:** Database and Auth.
- **OpenAI:** GPT-4o-mini (Logic), Whisper (STT), TTS-1 (Audio).

## Setup
1. `pip install -r requirements.txt`
2. Create `.env` file with credentials.
3. Run `python -m src.main`
